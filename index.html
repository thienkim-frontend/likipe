Vår arbetsprocess

Våra värderingar har funnits med oss ända sedan start; resultat, ansvar, enkelhet och glädje. Vi tror på att förmedla våra värderingar tydligt ger bästa kvalitet i ett utfört arbete som i sin tur ger nöjda kunder.

För att kunna kundanpassa webbsystemet och veta exakt vad kunden vill ha, krävs det tillräckligt med dokumentation, vi hjälper till att ta fram dessa dokument genom att ha workshops. Databaser

För att få tillgång till rätt data snabbt och enkelt, och framförallt utföra komplicerade databeräkningar utan att förlora mycket prestanda i systemet, använder man sig utav databaser. Genom databaser blir det även mycket mer flexibelt att jobba med själva informationen eftersom det lagras strukturerat via en databashanterare.

Idag finns det olika lösningar på hur en databas kan vara uppbyggd. De vanligaste som används idag är MySQL, PostgreSQL, Oracle, MongoDB, CouchDB och Redis. Varje lösning bär med sig sina fördelar och nackdelar. Idag är det MySQL som erbjuds som standard hos de flesta webbhotellen.

Teknik

Vi använder verktyg som Database .NET för att hantera flera databaser samtidigt och även Linux PHP som är mycket effektivt då det stödjer en mängd olika databaser.

Hur gör vi det

Utöver kraftfulla verktyg ser vi till att databaser förblir säkra och optimerade. Ett exempel för att säkra databasen är att vi, bland annat, använder multipla servrar med separata databaser. Medan all kommunikation går via en huvuddatabas så finns det en sekundär databas som konstant speglar av huvuddatabasen via synkronisering. I samband med att se till att minimala anropningar används ser vi till att databasen förblir säkrad och samtidigt optimerade.
